MacOS升级High sierra提示需要接通电源解决

ZMarvel
2018-05-06
操作系统
MacOS, 原创

由于笔记本的电池坏了，一直都是直接连接电源适配器使用的，今天升级High Sierra提示需要连接电源，google搜索发现reset SMC可以解决,但过程复杂，不太靠谱，访问tonymacx86后发现可以通过命令行升级，使用命令行升级完美解决。

升级high sierra提示：
“Your computer is not connected to a power source / You must plug in and connect your power adapter before installing.

通过命令行升级解决：

Open /Applications/Utilities/Terminal and type the following:

Code (Text):
/Applications/Install\ macOS\ High\ Sierra.app/Contents/Resources/startosinstall

参考来源：https://www.tonymacx86.com/threads/update-directly-to-macos-high-sierra.232707/
