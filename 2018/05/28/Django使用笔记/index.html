Django使用笔记

ZMarvel
2018-05-28
软件开发
Python, 原创

记录Django使用过程中的一些笔记。

1、根据数据库表生成对应模型，后边可以跟上需要生成模型的表格名称，默认生成所有table的model

> python3 manage.py inspectdb hosts services notes > rockroll/models.py

2、Django根据模型生成数据库表格，建立模型以后使用：

> python manage.py makemigrations
> python manage.py migrate

2、python admin使用

> python manage.py createsuperuser
  Username:
  python manage.py createsuperuser

  Email address: sbuchanan@codeschool.com
  Password: ...
  Password (again): ...
  Super user created successfully.
