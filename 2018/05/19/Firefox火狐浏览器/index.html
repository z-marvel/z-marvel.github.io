同步服务器切换

为了解决的国内用户连接火狐全球同步服务器慢的问题，火狐中国版推出了 本地服务 和 全球服务 两套服务。
安装Ubuntu默认安装的Firefox是全球版， 不具有切换 本地服务 和 全球服务 的功能。并不需要重新安装中国版火狐浏览器，只需要下载中国版火狐浏览器，运行一次i，重新打开Ubuntu自带的火狐浏览器就会发现可以切换 本地服务 和 全球服务 了 ，并且默认是本地服务。

安装Adobe flash

命令行版本

sudo apt-get update
sudo apt-get install flashplugin-installer

手工安装

打开adobe flash player下载页
选择下载tar.gz包
解压tar.gz包
readme.txt里边有安装和卸载说明
我是直接：
复制libflashplayer.so到浏览器插件目录下，对于ubuntu系统，路径为/usr/lib/firefox-addons/plugins/
即可
