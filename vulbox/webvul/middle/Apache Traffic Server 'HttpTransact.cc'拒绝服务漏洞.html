漏洞描述
漏洞名称 Apache Traffic Server 'HttpTransact.cc'拒绝服务漏洞
详细描述 Apache Traffic Server是可扩展的HTTP/1.1合规缓存代理服务器。

Apache Traffic Server 5.0.0-5.1.1版本在'HttpTransact.cc'的实现上存在拒绝服务漏洞，远程攻击者利用此漏洞可使受影响进程崩溃。

<*来源：Matthew Daley
*>
解决方案 厂商补丁：

Apache Group
------------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：

https://docs.trafficserver.apache.org/en/latest/api/HttpTransact_8cc_source.html
