漏洞描述
漏洞名称 Apache 'mod_imap' Referer 跨站脚本漏洞
详细描述 Apache HTTP Server是Apache软件基金会的一个开放源码的网页服务器，可以在大多数计算机操作系统中运行。

Apache httpd 1.3.35-dev 之前版本和 Apache httpd 2.0.x 的 2.0.56-dev 之前版本的 mod_imap 模块中存在跨站脚本攻击漏洞，远程攻击者可以在使用图片映象时通过 Referer 注入任意 web 脚本或 HTML。
解决方案 厂商补丁：

Apache Group
------------
目前厂商已经发布升级补丁解决此安全问题，补丁获取链接：

http://archive.apache.org/dist/httpd/
