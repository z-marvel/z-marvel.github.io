漏洞名称 Apache Tomcat Documentation Sample 应用程序多个跨站脚本攻击漏洞
详细描述 Apache Tomcat是一个流行的开放源码的JSP应用服务器程序。

Tomcat 的appdev/sample/web/hello.jsp example应用程序中存在多个跨站脚本攻击漏洞。远程攻击者可以借助test参数和其他未明向量，注入任意的web脚本或HTML。
解决方案 厂商补丁：

Apache Tomat
------------
[1]目前厂商已经发布升级版本修复此安全问题，建议升级apache tomcat 至4.1.37或之后版本：
http://tomcat.apache.org/security-4.html
[2]目前厂商已经发布升级版本修复此安全问题，建议升级apache tomcat 至5.5.24, 5.0.SVN或之后版本：
http://tomcat.apache.org/security-5.html
[3]目前厂商已经发布升级版本修复此安全问题，建议升级apache tomcat 至6.0.11或之后版本：
http://tomcat.apache.org/security-6.html
