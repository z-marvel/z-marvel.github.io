漏洞描述
漏洞名称 WordPress wp-includes/comment.php远程SQL注入漏洞
详细描述 WordPress是一款免费的论坛Blog系统。

WordPress在实现上存在SQL注入漏洞，攻击者可利用这个漏洞控制应用程序，访问或修改数据，攻击数据库本身。

此漏洞源于wp-includes/comment.php脚本没有正确检查过滤用户输入就用于SQL查询，导致通过注入任意SQL代码操纵SQL查询。利用此漏洞需要“Author-level”的访问权限。
解决方案 厂商补丁：

WordPress
---------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：

http://wordpress.org/
