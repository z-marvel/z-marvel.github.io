漏洞名称 Apache Tomcat 参数错误信息泄露漏洞
详细描述 Apache Tomcat是一个流行的开放源码的JSP应用服务器程序。

Apache Tomcat当参数处理过程中发生异常时，会处理错误请求的上下文参数，可能允许远程攻击者取得敏感信息，这表现在处理过程中断开，以触发例外。
解决方案 厂商补丁：

Apache Group
------------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：

http://jakarta.apache.org/tomcat/index.html
