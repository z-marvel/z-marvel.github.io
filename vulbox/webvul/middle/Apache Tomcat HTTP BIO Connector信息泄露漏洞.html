漏洞描述
漏洞名称 Apache Tomcat HTTP BIO Connector信息泄露漏洞
详细描述 Apache Tomcat是一个流行的开放源码的JSP应用服务器程序。

Apache Tomcat在HTTP BIO连接器的实现上存在安全漏洞，攻击者可利用此漏洞获取敏感信息。

<*来源：Brad Piles
*>
解决方案 厂商补丁：

Apache Group
------------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：

http://jakarta.apache.org/tomcat/index.html
