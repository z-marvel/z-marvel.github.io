漏洞描述
漏洞名称 检测到模板Apache Server Status敏感信息泄露
详细描述 Apache的日志又多又杂，如果靠分析日志或者查看服务器进程来监视Apache运行状态的话，比较繁冗。在Apache 1.3.2及以后的版本中就自带一个查看Apache状态的功能模块server-status。
如果这个页面对公网开放，就会存在一些安全隐患。
例如任何人口可以看到谁在访问网站，甚至包括一些本来隐藏的管理页面。
解决方案 1、在需要开启此模块的情况下，在配置文件中，配置Deny from（表示禁止的访问地址），或者配置Allow from（表示允许的地址访问）来进制其他攻击者访问此链接
2、不需要开启此模块时，可以关闭此功能模块，注释掉httpd.conf或者extra/httpd-info.conf中有LoadModule status_module modules/mod_status.so即可。
