漏洞描述
漏洞名称 Apache mod_proxy_ajp信息泄露漏洞
详细描述 Apache HTTP Server是一款流行的Web服务器。

Apache服务器的mod_proxy_ajp模块在处理畸形的POST请求时存在错误，远程攻击者可以通过提交特制的HTTP请求泄露其他用户请求相关的响应数据。

<*来源：Sander de Boer （sander.de.boer@eo.nl）

链接：http://secunia.com/advisories/34827/
https://issues.apache.org/bugzilla/show_bug.cgi?format=multiple&id=46949
*>
解决方案 厂商补丁：

Apache Group
------------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：

http://www.apache.org/dist/httpd/patches/apply_to_2.2.11/PR46949.diff
