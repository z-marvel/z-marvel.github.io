漏洞描述
漏洞名称 ibm websphere_application_server File Transfer servlet 配置错误漏洞
详细描述 IBM WebSphere Application Server是一个完善的、开放的Web应用服务器，它是IBM电子商务应用架构的核心。
IBM WebSphere Application Server (WAS) 6.1.0.19之前的6.1.x版本中的系统管理/报告组件里的File Transfer servlet的安装进程没有激活安全版本，这使得远程攻击者可以借助未明向量，获得敏感信息。
解决方案 厂商补丁：

IBM
