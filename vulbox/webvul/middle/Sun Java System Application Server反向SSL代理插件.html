漏洞名称 Sun Java System Application Server反向SSL代理插件
详细描述 Sun Java System Application Server Standard Edition 7 2004Q2、Application Server Enterprise Edition 8.1 2005Q1和Sun ONE Application Server 7 Standard Edition的反向SSL代理插件中的未指定漏洞(用在多个Web服务器中)，远程攻击者可以执行中间人(MITM)攻击并"泄漏数据隐私"。
解决方案 目前厂商已经发布了升级补丁以修复这个安全问题，补丁下载链接：

https://sunsolve.sun.com/searchproxy/document.do?assetkey=1-26-102012-1
