漏洞描述
漏洞名称 Cherokee Web Server GET request拒绝服务漏洞
详细描述 Cherokee Web Server是一个灵活，速度非常快，轻量级的Web服务器。 Cherokee Web Server 的GET request存在拒绝服务漏洞。远程攻击者可以借助URI中的MS-DOS预留字，比如AUX预留字，导致引起拒绝服务攻击(后台程序崩溃)。
解决方案 建议升级Cherokee产品至最新版本，请到厂商的主页下载：
http://www.cherokee-project.com/
威胁分值 4
危险插件 否
