漏洞名称 PHP Mysqlnd扩展php_mysqlnd_ok_read()函数信息泄露漏洞
详细描述 本次扫描方式是版本扫描，可能出现误报。

PHP是广泛使用的通用目的脚本语言，特别适合于Web开发，可嵌入到HTML中。

PHP所使用的Mysqlnd扩展用于处理与mysql服务器的通讯和实现mysql协议。该扩展的php_mysqlnd_ok_read()函数中存在信息泄露漏洞：

/* There is a message */
if (packet->header.size > (size_t) (p - buf) && (i = php_mysqlnd_net_field_length(&p))) {
packet->message = mnd_pestrndup((char *)p, MIN(i, buf_len - (p - begin)), FALSE);
packet->message_len = i;
} else {
packet->message = NULL;
}

上述代码从网络缓冲区读取消息长度并将消息拷贝到新分配的缓冲区中。这个拷贝操作的实现是安全的，因为确保了仅拷贝了缓冲区中可用的字节数。但 message_len字段错误的设置为了网络所提供的值，导致所有处理消息和报文结构中message_len信息的函数中都存在信息泄露。

PHP
