漏洞描述
漏洞名称 Sun Java System Application Server远程安装路径泄露漏洞
详细描述 Sun Java System Application Server是与J2EE平台兼容的应用服务器。

Java System Application Server不正确过滤用户提交的请求，远程攻击者可以利用这个漏洞获得服务程序的安装路径信息。

提交如下类似的请求给Sun-Java-App-Server PE 8.0：

http://127.0.0.1:8080////
http://127.0.0.1:8080////CON

服务器会返回包含安装路径信息的错误信息。攻击者可以利用这个信息进一步对系统进行攻击。
解决方案 Sun
---
目前厂商已经发布了升级补丁以修复这个安全问题，补丁下载链接：

http://wwws.sun.com/software/products/appsrvr/home_appsrvr.html
