漏洞描述
漏洞名称 Apache Web服务程序文件描述符泄露漏洞
详细描述 Apache是一款流行的开放源代码httpd服务程序。

Apache由于子进程不正确继承文件描述符，远程攻击者可以利用这个漏洞获得敏感信息，如访问敏感日志信息。

目前没有详细漏洞细节。
解决方案 Apache
------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：

Apache Software Foundation Upgrade Apache httpd 2.0.45
http://www.apache.org/dist/httpd/
