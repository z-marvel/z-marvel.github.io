漏洞名称 Code-Crafters Ability Mail Server IMAP FETCH Request 远程拒绝服务漏洞
详细描述 Code-Crafters Ability Mail Server 2.70版本之前的版本中存在未明漏洞。远程攻击者借助一个IMAP4 FETCH指令,造成拒绝服务(后台程序崩溃)。
解决方案 目前厂商已经发布了升级补丁以修复这个安全问题，补丁下载链接：
http://download.code-crafters.com/ams.exe
