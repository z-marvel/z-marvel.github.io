漏洞名称 Jetty未明的服务拒绝漏洞
详细描述 Jetty HTTP Server 4.2.19以前版本的HttpRequest.java存在漏洞。远程攻击者借助具有超长Content-Length的HTTP请求导致服务拒绝（内存的使用情况和应用程序崩溃）。
解决方案 厂商补丁：
Jetty:
------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：
http://sourceforge.net/projects/jetty/
