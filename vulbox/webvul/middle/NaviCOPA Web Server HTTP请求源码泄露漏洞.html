漏洞描述
漏洞名称 NaviCOPA Web Server HTTP请求源码泄露漏洞
详细描述 NaviCOPA Web Server是一款安装在Windows系统上的Web服务器，可以自动配置HTTP访问。

如果远程攻击者在提交给NaviCOPA服务器的HTTP请求中附带了“20”字符的话，就可以从服务器返回中读取PHP等脚本的源码。
