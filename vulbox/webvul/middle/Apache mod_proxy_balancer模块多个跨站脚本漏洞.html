漏洞描述
漏洞名称 Apache mod_proxy_balancer模块多个跨站脚本漏洞
详细描述 Server 2.2.7版权及之前的2.2.x所有版本的mod_proxy_balancer模块存在跨站请求伪造漏洞，当在windows系统上运行时，允许远程攻击者通过一个长未明向量获取特权。
解决方案 厂商补丁：

Apache
------------
目前厂商已经发布升级版本修复此安全问题，建议升级Apache产品至2.2.8版本或之后版本，请到厂商的官方页面下载升级版本：
http://archive.apache.org/dist/httpd/
