漏洞描述
漏洞名称 Apache HTTP服务器403 Error页面跨站脚本漏洞
详细描述 Apache HTTP Server是一款流行的Web服务器。
如果Apache服务器使用Forbidden 403默认页面的话，则用户在该服务器上请求特制的URL触发403 Error错误后就可能导致在用户浏览器会话中执行跨站脚本。

*>
