漏洞描述
漏洞名称 Apache HTTP Server Windows Share PHP File Extension Mapping 信息泄露漏洞
详细描述 Apache HTTP服务器可能存在存在信息泄露漏洞。当使用smbfs服务共享Windows和Linux之间的文件时，Apache HTTP服务器，远程攻击者可以借助未被AddType处理的"\\"获得未处理得内容，如.php程序的源文件。
解决方案 厂商补丁：

Apache Group
------------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载1.3.27或之后版本：

http://httpd.apache.org/download.cgi
