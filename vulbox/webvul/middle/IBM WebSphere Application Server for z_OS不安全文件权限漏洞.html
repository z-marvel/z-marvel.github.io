漏洞描述
漏洞名称 IBM WebSphere Application Server for z/OS不安全文件权限漏洞
详细描述 IBM Websphere应用服务器以Java和Servlet引擎为基础，支持多种HTTP服务，可帮助用户完成从开发、发布到维护交互式的动态网站的所有工作。

WebSphere应用服务器的System Management/Repository组件中存在信息泄露漏洞，在z/OS操作系统上部署新的应用会导致以不安全的文件权限将其保存到文件系统上。远程攻击者可以通过读取这些文件获得敏感信息。

*>
