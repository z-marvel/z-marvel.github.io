漏洞描述
漏洞名称 IBM WebSphere应用服务器跨站请求伪造漏洞
详细描述 IBM WebSphere Application Server (WAS)是由IBM遵照开放标准，例如Java EE, XML 还有Web Services，开发并发行的一种应用服务器。IBM WebSphere Application Server存在跨站请求伪造漏洞，攻击者构建恶意链接，诱使登录用户点击解析，可以登录用户上下文执行管理操作。
解决方案 IBM Websphere Application Server 8.0.0.1已经修复此漏洞，建议用户下载使用：http://www-01.ibm.com/support/docview.wss?uid=swg24030916
