漏洞名称 IBM WAS跨站请求伪造漏洞
详细描述 IBM WebSphere Application Server 是一个完善的、开放的Web应用服务器，是IBM电子商务应用架构的核心。
IBM WebSphere Application Server (WAS) 6.1.0.35之前的6.1版本，以及7.0.0.13之前的7.0版本的管理控制台中存在跨站请求伪造漏洞。远程攻击者可以借助未知向量劫持未明受害者的认证。
解决方案 目前厂商已经发布了升级补丁以修复此安全问题，补丁获取链接：
http://www-01.ibm.com/support/docview.wss?uid=swg1PM23874
