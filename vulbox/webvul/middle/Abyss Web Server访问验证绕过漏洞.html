漏洞描述
漏洞名称 Abyss Web Server访问验证绕过漏洞
详细描述 Abyss Web Server是一宽免费的小型的WEB服务程序。

Abyss Web Server在对密码保护目录访问处理存在问题，远程攻击者可以利用这个漏洞绕过限制，未验证访问服务器资源。

此问题只存在在linux系统，而当保护的文件夹在FAT32系统上，这个条件不是很常见，但由于Abyss配置方便，使用Linux提供共享一个FAT32目录也经常存在。当管理员在Linux上运行Abyss Web服务器，并使用密码对FAT32目录进行防护时，攻击者可以简单的提交点或者HTTP编码的点(2e)来绕过验证，另外字符空格(' ', 20)和':' (3a)同样会引起此问题。
解决方案 Aprelium Technologies
---------------------
目前厂商已经在最新版本的软件中修复了这个安全问题，请到厂商的主页下载：

Aprelium Technologies Upgrade Abyss Web Server 1.2
http://www.aprelium.com/abyssws/download.php
