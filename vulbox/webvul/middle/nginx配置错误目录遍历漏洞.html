详细描述 nginx是一款使用非常广泛的高性能web服务器。

nginx的某些配置方式存在错误，可被远程攻击者利用造成目录遍历。

<*来源：@Sebug.net

链接：http://sebug.net/vuldb/ssvid-26117
*>
解决方案 临时解决方法：

使用如下配置
location /paper {
alias /home/wwwroot/paper;
或
location /paper/ {
alias /home/wwwroot/paper/;
