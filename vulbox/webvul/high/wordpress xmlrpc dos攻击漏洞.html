漏洞描述
漏洞名称 wordpress xmlrpc dos攻击漏洞
详细描述 WordPress是一种使用PHP语言开发的博客平台，用户可以在支持PHP和MySQL数据库的服务器上架设自己的网志。也可以把 WordPress 当作一个内容管理系统（CMS）来使用。
WordPress的xmlrpc.php在处理xml请求时处理不当导致DoS漏洞。
受影响系统：
WordPress 3.9.x-3.9.1
WordPress 3.8.x-3.8.3
WordPress 3.7.x-3.7.3
WordPress 3.6.x
WordPress 3.5.x
不受影响系统：
WordPress 3.9.2
WordPress 3.8.4
WordPress 3.7.4
