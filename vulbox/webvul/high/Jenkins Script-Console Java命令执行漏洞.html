漏洞名称 Jenkins Script-Console Java命令执行漏洞
详细描述 Jenkins是基于Java开发的一种持续集成工具，用于监控持续重复的工作，功能包括：
1、持续的软件版本发布/测试项目。
2、监控外部调用执行的工作。
Jenkins若安装了脚本命令执行插件Script Console，恶意用户即可通过模块执行操作系统命令从而进一步控制服务器。
解决方案 1、更改配置禁止未授权访问，对用户进行身份认证。
2、卸载Script Console插件或禁止未认证用户访问Script Console插件。
