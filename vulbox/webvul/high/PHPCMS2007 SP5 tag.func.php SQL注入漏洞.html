漏洞描述
漏洞名称 PHPCMS2007 SP5 tag.func.php SQL注入漏洞
详细描述 Phpcms网站管理系统是国内主流CMS系统之一。

漏洞文件/formguide/include/tag.func.php：
global $db,$MOD,$MODULE;
$formid = (!isset($formid)||$formid=='') ? 0 : $formid;
$query = "SELECT * FROM ".TABLE_FORMGUIDE." WHERE formid=$formid limit 1";
$r = $db->get_one($query);
变量formid未赋值且过滤不严，导致SQL注入产生。
解决方案 厂商补丁：
PHPCMS
-------
目前厂商已经提供补丁或者升级程序，我们建议使用此软件的用户从厂商的主页获取最新版本：
http://www.phpcms.cn
