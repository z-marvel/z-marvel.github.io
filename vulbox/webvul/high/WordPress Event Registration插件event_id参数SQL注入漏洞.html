漏洞描述
漏洞名称 WordPress Event Registration插件event_id参数SQL注入漏洞
详细描述 WordPress是一款免费的论坛Blog系统。

当regevent_action设置为register的时候，WordPress没有正确地过滤提交给events页面的event_id参数就在SQL查询中使用，这可能导致SQL注入攻击。
解决方案 厂商补丁：

WordPress
---------
目前厂商还没有提供补丁或者升级程序，我们建议使用此软件的用户随时关注厂商的主页以获取最新版本：

http://wordpress.org/
