漏洞名称 应用服务器glassfish任意文件读取漏洞
详细描述 GlassFish 是用于构建 Java EE 5应用服务器的开源开发项目的名称。它基于 Sun Microsystems 提供的 Sun Java System Application Server PE 9 的源代码以及 Oracle 贡献的 TopLink 持久性代码。该项目旨在促进 Sun 和 Oracle 工程师与社区之间的交流，它将使得所有开发者都能够参与到应用服务器的开发过程中来。
由于在java语言中会把"%c0%ae"解析为"uC0AE"，最后转义为ASCCII字符的"."，所以通过"%c0%ae%c0%ae/"（../）可以回朔目录，导致可以遍历整个系统的全部目录结构，以及读取下载任意文件内容，从而导致整个系统沦陷。
解决方案 GlassFish
--------
目前厂商已经发布最新版本。
请到官方下载最新版进行升级。
