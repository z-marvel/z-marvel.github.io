漏洞描述
漏洞名称 HDwiki5.0任意文件上传漏洞
详细描述 hdwiki是互动百科开发的一款免费建站，自助建站程序。

但是HDwiki中某些上传功能存在安全漏洞，通过一些数据即可绕过上传限制，最终控制远程站点.
解决方案 厂商解决方案:

修改file.class.php文件里的函数function uploadfile(）
if(substr($attach['type'],0,6)!='image/' || !in_array(file::extname($attach['name']), array('jpg','jpeg','bmp','gif','png'))) {
$result['msg'] ='格式错误';
return $result;
}
