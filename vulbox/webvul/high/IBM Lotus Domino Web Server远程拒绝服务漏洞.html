漏洞描述
漏洞名称 IBM Lotus Domino Web Server远程拒绝服务漏洞
详细描述 Lotus Domino/Notes是集电子邮件、文档数据库、快速应用开发技术以及Web技术为一体的电子邮件与群集平台。

Lotus Domino Web Server在处理特定URL请求时存在漏洞，远程攻击者可能利用此漏洞对服务器执行拒绝服务攻击。

在某些环境下，如果访问了引用特定文件的URL就可能导致Lotus Domino Web服务器崩溃，造成拒绝服务。
