漏洞描述
漏洞名称 检测到目标站点存在 jQuery XSS with $(location.hash) 漏洞
详细描述 jQuery 是一个 JavaScript 库。
版本低于 1.6.3 的 jQuery 在过滤用户输入数据时，所使用的正则表达式存在缺陷，故而存在跨站脚本漏洞。当使用 location.hash 来选取元素时，远程攻击者可通过一个精心制作的标签注入任意 web 脚本或者 HTML 代码。
解决方案 1、将使用的jQuery库中的
quickExpr = /^(?:[^<]*(<[\w\W]+>)[^>]*$|#([\w\-]*)$)/，
修改为
quickExpr = /^(?:[^#<]*(<[\w\W]+>)[^>]*$|#([\w\-]*)$)/。
2、升级jQuery库：
https://code.jquery.com/
