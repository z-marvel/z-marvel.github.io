详细描述 Phpcms网站管理系统是国内主流CMS系统之一。

导致漏洞产生的代码：
phpcms中vote/vote.php第22行
$optionids = is_array($op) ? implode(',',$op) : $op;
...
$db->query("UPDATE ".TABLE_VOTE_OPTION." SET number = number+1 WHERE optionid IN ($optionids) ");
