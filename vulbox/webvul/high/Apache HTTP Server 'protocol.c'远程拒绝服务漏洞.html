漏洞名称 Apache HTTP Server 'protocol.c'远程拒绝服务漏洞
详细描述 Apache HTTP Server是Apache软件基金会的一个开放源代码的网页服务器。

Apache HTTP Server 2.2.29/2.4.12在实现上存在空指针间接引用，成功利用后可造成受影响应用崩溃。

<*来源：Nicholas Lemonias

链接：http://www.securityfocus.com/archive/1/535273
*>
解决方案 厂商补丁：

Apache Group
------------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：

http://httpd.apache.org/
