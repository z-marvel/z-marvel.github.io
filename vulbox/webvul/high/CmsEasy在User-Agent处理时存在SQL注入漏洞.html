漏洞描述
漏洞名称 CmsEasy在User-Agent处理时存在SQL注入漏洞
详细描述 Cmseasy一款基于 PHP+Mysql 架构的网站内容管理系统，也是一个 PHP 开发平台。 采用模块化方式开发，功能易用便于扩展，可面向大中型站点提供重量级网站建设解决方案。
由于Cmseasy在处理搜索引擎Agent内容时，没有对$_SERVER中的内容进行过滤，从而造成SQL注入漏洞，远程攻击者可以利用此漏洞获取敏感信息及非法操作数据库。
CmsEasy版本小于等于5_5_0_20140605的都很可能存在此漏洞。
解决方案 Cmseasy
-------
http://www.cmseasy.cn/
厂商到目前尚未提供最新版程序，请关注官方补丁及升级程序，及时下载对应补丁或者升级程序到最新版。
