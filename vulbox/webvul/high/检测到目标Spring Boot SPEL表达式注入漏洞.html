漏洞名称 检测到目标Spring Boot SPEL表达式注入漏洞
详细描述 在当今Java EE 开发中，Spring 框架是当之无愧的王者。而Spring Boot 是Spring 主推的基于“习惯优于配置”的原则，让你能够快速搭建应用的框架，从而使得Java EE 开发变得异常简单。Spring Boot是Spring的一个核心子项目，已经得到广泛使用。

由于在用户采用了Spring Boot启动Spring MVC项目后，Spring Boot的默认异常模板在处理异常信息的时候可导致SPEL表达式注入并执行。

影响版本：Spring Boot 1.1 1.2 - Spring Boot 1.3.0
解决方案 修复方法：
方法1：全局添加异常处理页面。
方法2：升级Spring Boot版本至1.3.1或以上版本。
