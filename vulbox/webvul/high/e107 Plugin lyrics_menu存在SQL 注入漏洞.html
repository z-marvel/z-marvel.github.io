漏洞描述
漏洞名称 e107 Plugin lyrics_menu存在SQL 注入漏洞
详细描述 e107 Lyrics Plugin是一个e107内容管理系统的一个歌曲歌词管理插件。 Lyrics (lyrics_menu) plugin 0.42版本中的lyrics_song.php存在SQL注入漏洞。远程攻击者可以借助l_id参数，执行任意的SQL指令。
解决方案 厂商补丁
e107
--------
目前厂商还没有提供补丁或者升级程序，建议使用此软件的用户随时关注厂商的主页以获取最新版本：
http://e107.org/
