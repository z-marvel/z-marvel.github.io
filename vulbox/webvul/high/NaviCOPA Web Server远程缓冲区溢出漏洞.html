漏洞描述
漏洞名称 NaviCOPA Web Server远程缓冲区溢出漏洞
详细描述 NaviCOPA Web Server是一款安装在Windows系统上的Web服务器，可以自动配置HTTP访问。

NaviCOPA Web Server在处理超长请求时存在缓冲区溢出漏洞，远程攻击者可以通过向服务器发送超长请求触发缓冲区溢出，导致执行任意指令。
解决方案 InterVations
------------
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：
http://www.navicopa.com/download.html
