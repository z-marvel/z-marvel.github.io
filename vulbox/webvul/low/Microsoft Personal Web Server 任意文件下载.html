漏洞描述
漏洞名称 Microsoft Personal Web Server 任意文件下载
详细描述 微软的Personal WebServer存在一个目录遍历漏洞。攻击者可以读取远程系统的任何文件。可能会查看 Web 服务器（在 Web 服务器用户的许可权限制下）上的任何文件（例如，数据库、用户信息或配置文件）的内容。
利用的样本：
GET /....../FILE HTTP/1.0
例如：GET ........../autoexec.bat
解决方案 微软已经为此发布了一个安全公告（MS99-010）以及相应补丁程序：
http://www.microsoft.com/technet/security/bulletin/MS99-010.asp
