漏洞名称 检测到目标URL启用了 Microsoft ASP.NET 调试
详细描述 Microsoft ASP.NET 很容易受到信息泄露攻击。攻击者可以发送一个通知是否支持调试支持的恶意请求。
攻击者可以利用 DEBUG 动词来发送恶意请求。
解决方案 如果要在 ASP.NET 中禁用调试，请编辑您的 web.config 文件，使它含有下列属性：
<compilation

debug="false"
/>
