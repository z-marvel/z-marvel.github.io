kali更新内核

ZMarvel
2019-02-10
操作系统
Linux, 原创

安装vbox extension pack需要安装Linux header，安装Linux header值钱啊需要更新Linux内核：
更新内核：
apt install linux-image-amd64
安装头文件：
apt instal linux-header-amd64
或者：apt instlall linux-header-uname -r
注意：
对于第种方法，更新内核以后需要重启才可以安装头文件，或者手动选择头文件安装。
